# Black Jack Towing & Recovery Website

Official website for Black Jack Towing & Recovery.

## Tech Stack

- React
- Vite
- Firebase Hosting
- GitHub

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Deploy to Firebase:

```bash
firebase deploy
```
